## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 오픈월드 탐색 + 사이드 스토리 퀘스트 진행 + 대화 기반 내러티브. 캐릭터를 조작하여 실내 환경(달빛 전당포)을 탐색하고, NPC와 대화하며 스토리를 진행하는 구조. 상호작용 오브젝트("열기" 프롬프트)가 있어 환경 탐색 요소도 포함.
- UI / 연출 인상: 원신/붕괴 스타레일 계열의 서브컬쳐 UI 문법을 따름. 미니맵, 4인 파티 초상화(우측), HP/스태미나 바, 스킬 버튼(R/E/Q) 배치. 대화 장면에서는 UI를 최소화하고 캐릭터 반신 연출에 집중. 전체적으로 세련되고 고품질의 실시간 렌더링. 공생 테스트 버전 워터마크 표시.
- 메타 시스템 흔적: 4인 파티 편성 시스템(우측 캐릭터 슬롯), 레벨 시스템(Lv.18), HP 수치(3151/3151), 스킬 쿨다운 시스템 확인. 수집형 캐릭터 기반 편성 메타가 핵심으로 보임.

## 평가
- 트렌드 적합성: 높음. 원신으로 시작된 오픈월드 서브컬쳐 RPG 트렌드의 직접적 후속 주자. 달빛 전당포라는 독자적 세계관 요소로 차별화 시도.
- 프로덕션 퀄리티: 매우 높음. 3D 캐릭터 모델링, 실내 환경 라이팅, 대화 연출 모두 AAA급 모바일 수준. 캐릭터 디자인이 개성적이며 애니메이션 품질 우수.
- 게임 깊이: 중간-높음. 영상이 사이드 스토리 후반부 플레이라 전투 시스템은 미확인이나, 스토리 탐색/대화/환경 인터랙션 등 다층적 콘텐츠 구조 암시. 4인 파티 + 스킬 시스템으로 전투 깊이 기대.
- 아트 트렌드: 동양 판타지(중화풍) + 현대적 서브컬쳐 스타일 혼합. 전당포라는 공간에 나무/종이 소재의 일본-중국 절충 인테리어. 캐릭터는 고딕/펑크 요소가 가미된 서브컬쳐 디자인.
- 종합 코멘트: NetEase의 오픈월드 서브컬쳐 RPG 기대작으로, 초자연 판타지 세계관과 높은 프로덕션 퀄리티가 인상적. 공생 테스트 단계임에도 완성도 높은 연출과 UI를 보여주며, 원신/명조 계열 시장에서 경쟁력 있는 후보.

## 심층 분석

### 영상 심층 시청
- 초/중/후반부 캡처 기반 게임플레이 진행 흐름: 초반(~1분)은 달빛 전당포 실내에서 망원경/고서적 등 오브젝트 탐색 + NPC "나나리"와의 대화로 스토리 진입. 중반(~5분)에서는 두 캐릭터(은발/적발)가 전당포 선반 앞에서 NPC "포르테"의 서사를 듣는 멀티 캐릭터 내러티브 연출. 후반(~9분30초)에서는 "오디티 제거하기" 전투 퀘스트가 시작되며, 실시간 3D 액션 전투(데미지 숫자 12, 노멀 타격, 스킬 쿨다운 8.9초)가 전개됨. 스토리 탐색 → 대화/내러티브 → 실시간 전투로 이어지는 3단 구조.
- 메타 시스템 (육성/수집/뽑기/스킬 트리 등): 가챠 기반 캐릭터 수집이 핵심 BM. 4인 파티 편성(우측 캐릭터 슬롯), 레벨 시스템(Lv.18), HP(3151/3151), 스킬 쿨다운(R/E/Q 키), 스태미나 시스템 확인. 영상 설명란에 따르면 주 BM은 가챠이며, 개발사는 가성비 좋은 BM이 될 것으로 전망. 카페 운영/펜트하우스 등 경영 콘텐츠도 포함.
- UI 흐름: 원신/명조 계열 서브컬쳐 UI 문법. 좌상단 미니맵, 우측 4인 파티 초상화, 하단 HP/스태미나 바, 우하단 스킬 버튼(R/E/Q). 대화 장면에서는 캐릭터명+대사 자막 중심으로 UI 최소화. 전투 시 데미지 팝업, 퀘스트 목표(좌상단), 채팅 입력(하좌) 표시. 공생 테스트 워터마크 및 UID 표시.
