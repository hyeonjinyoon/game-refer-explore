## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 1인칭 슈터 기반 야생동물 사냥(스코프 조준, 마취/실탄 선택) + 거점 건설/자원 관리(쿼터뷰 전략 맵). 사냥 미션 클리어 후 자원 획득 -> 기지 확장의 이중 루프 구조.
- UI / 연출 인상: 사냥 모드에서는 리얼리스틱한 자연환경 + FPS 스코프 UI(탄환 비행 연출 토글 있음). 기지 모드에서는 전형적 SLG 스타일 쿼터뷰에 자원 바 표시. 한국어 로컬라이징 완료. 전반적으로 높은 그래픽 수준.
- 메타 시스템 흔적: 자원 화폐 다종(700K/695K 등 복수 자원), 기지 내 다수 건물 슬롯, "황야 사냥" 모드명으로 보아 복수 사냥 모드/맵 존재 추정. 전형적 모바일 SLG 과금 구조 예상.

## 평가
- 트렌드 적합성: FPS+SLG 하이브리드는 최근 모바일 시장에서 주목받는 복합 장르. 사냥 테마로 차별화 시도.
- 프로덕션 퀄리티: 높음. 자연환경 렌더링과 동물 모델링이 모바일 기준 상당 수준. 기지 건설 씬도 깔끔.
- 게임 깊이: 사냥(액션) + 기지(전략) 이중 루프로 중간 이상의 깊이. 다만 각 요소의 개별 깊이는 전문 장르 대비 얕을 수 있음.
- 아트 트렌드: 리얼리스틱 자연환경 + 판타지 건축물 혼합. 주류 SLG 아트 방향과 유사하나 사냥/야생 테마로 시각적 차별점 확보.
- 종합 코멘트: FPS 사냥과 SLG 기지건설을 결합한 하이브리드 모바일 게임. 그래픽 퀄리티는 높으나, 두 장르의 결합이 유기적인지는 실제 플레이 확인 필요. 사냥 테마의 독특함이 강점.
