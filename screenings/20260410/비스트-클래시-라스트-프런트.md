## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 월드맵 탐험(비스트 포획) → 비스트 육성/강화(전투력 상승) → 기지 건설·확장 → 전투/퀘스트 순환. 비스트를 중심 축으로 SLG 루프를 구성.
- UI / 연출 인상: 가로 모드, 상단 재화 바(500K~700K 다종 재화), 비스트 상세 화면에서 3D 모델 + 전투력/스탯 표시, 기지 화면은 아이소메트릭 뷰. UI 전반적으로 중국 SLG 문법을 따르며 한글화 완료. "아무 곳이나 눌러 계속하기" 등 전형적 SLG 연출.
- 메타 시스템 흔적: VIP 시스템 확인, 다운로드 선물(사전등록 보상 추정), 상점/인벤토리 메뉴, 자이언트(비스트) 수집 시스템, 월드맵 모드 존재. 스탯·전투력 수치가 매우 높아(352,665 / 413,638) 전형적 수치 인플레이션 기반 메타.

## 평가
- 트렌드 적합성: 공룡/비스트 테마 SLG는 기존 좀비·중세 SLG 대비 차별화 시도. 수집+육성 요소 결합은 트렌드에 부합하나, SLG 기본 구조 자체는 보수적.
- 프로덕션 퀄리티: 비스트 3D 모델링 품질 양호, 기지·월드맵 그래픽 수준 중상. 전반적 UI 정리도 준수하나 중국 SLG 특유의 정보 과밀 경향 존재.
- 게임 깊이: 비스트 수집/육성 + 기지 건설 + 월드맵 탐험 등 다층적 시스템 보유. 그러나 영상 내 전투 장면 부재로 실제 전투 깊이는 미확인.
- 아트 트렌드: 포스트 아포칼립스 + 공룡/비스트 조합. 세미 리얼리스틱 스타일로 남성 타겟 SLG 시장에 맞춤. 비스트 디자인(강암각룡 등)은 독특하나 전체 아트 방향은 기존 SLG와 유사.
- 종합 코멘트: 공룡/비스트 테마로 SLG 레드오션에서 차별화를 시도한 작품. 수집·육성 루프가 SLG에 잘 녹아든 편이나, 기본 골격은 기존 중국 SLG 문법을 크게 벗어나지 않음. 3분 남짓의 짧은 영상으로 전투 시스템 등 핵심 콘텐츠 확인 불가.
- 선정 여부: TOP3
- 최종 ELO: 1534

## 심층 분석

### 영상 심층 시청
- 초/중/후반부 캡처 기반 게임플레이 진행 흐름: 초반(~18초)은 월드맵 탐험 장면으로, 아이소메트릭 뷰에서 비스트(공룡)와 군용 장비가 배치된 야생 필드를 이동하며 비스트를 포획하는 단계. 중반(~93초)은 저격 스코프(자이언트 마취 소총) 뷰로 전환되어 비스트를 1인칭 슈팅 방식으로 마취·포획하는 핵심 메커니즘 시연. 후반(~158초)은 기지 건설 화면으로 전투력 418,528 표시, 자이언트 센터 Lv.0 건설 완료, 토마토 오두막 건설 등 기지 확장 및 자원 관리 루프를 보여줌.
- 메타 시스템: VIP 시스템, 다운로드 선물(사전등록 보상), 상점/인벤토리, 자이언트(비스트) 수집 시스템, 월드맵 모드, 군단 시스템 확인. 전투력 수치 인플레이션 기반(418K+). 공식 사이트 충전 시스템(드래곤 코인=현금 쿠폰) 운영 중.
- UI 흐름: 가로 모드, 상단 재화 바(다종 재화), 하단 메뉴(월드 맵/자이언트/상점/인벤토리), 기지 화면은 아이소메트릭 뷰. 비스트 포획 시 1인칭 저격 스코프 전환이 특징적. 한글화 완료, 중국 SLG 문법 기반 UI.

- 회사 배경: Kingnet Technology (恺英网络, 킹넷네트워크)는 2008년 중국에서 설립, 2015년 상장된 게임 전문 기업. 20년 가까운 업력으로 게임 본업에만 집중하는 전략을 고수. 그라비티와 '노부나가의 야망 천하로의 길' 일본 퍼블리싱 계약을 체결하는 등 글로벌 퍼블리싱 역량 보유.
- 대표작: 전민기적MU, 오버로드:나자릭의 왕(4.5), 엉망 특공대(4.5), 리벤저:다크니스(4.4), 블러드 헌터: 도깨비전(3.9), 디바인: 신을 모시는 자(3.5), 세븐: 글로리 로드(3.2)
- 이전작 패턴: 중국 개발 SLG/RPG를 한국 시장에 현지화하여 퍼블리싱하는 패턴. IP 라이선스 활용(오버로드, 노부나가의 야망 등)과 자체 IP 개발을 병행. 평점 3.2~4.5로 편차가 있으나 전반적으로 중상 수준의 퀄리티 유지.
