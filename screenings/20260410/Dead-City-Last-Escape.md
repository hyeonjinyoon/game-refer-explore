## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 황무지(wasteland) 탐색 → 자원 수집(Search) → 전투(Battle) → 인벤토리 관리(Backpack/Safe box)의 서바이벌 루프. 탐색 시 스태미나 소모(5 stamina per search), 자원 포인트 품질(Blue→Green) 하락 시스템으로 탐색 횟수 제한. 전투 승리 시 EXP·Cash 획득.
- UI / 연출 인상: 세로 모바일 레이아웃. 2D 일러스트 기반 캐릭터(총기 소지 여성 캐릭터)와 적(뮤턴트 뱀 등). 상단에 캐릭터 스탯(EXP Lv.3, Life 468~471/488, SAN 45~50/60), 하단에 인터랙션 버튼. 전체적으로 인디 모바일 감성의 다크 톤 일러스트.
- 메타 시스템 흔적: Attribute 버튼(캐릭터 스탯 강화 추정), Backpack/Safe box 이원 인벤토리 시스템, 자원 품질 등급제(Blue/Green), 레벨링(Lv.3), SAN(정신력) 시스템으로 공포/스트레스 요소 반영.

## 평가
- 트렌드 적합성: 보통. 좀비/포스트아포칼립스 서바이벌은 모바일 시장에서 여전히 수요가 있으나, 차별화 요소가 뚜렷하지 않음. SAN(정신력) 시스템은 독특한 터치.
- 프로덕션 퀄리티: 낮음~보통. 2D 일러스트 기반 인디 수준. 애니메이션이 제한적이고 UI가 단순함. 텍스트 기반 서술(전투 결과 텍스트 박스)이 많음.
- 게임 깊이: 보통. 자원 관리(스태미나·품질 하락)와 인벤토리 관리가 전략적 판단을 요구하나, 전투 자체는 자동/간략 처리로 깊이가 제한적.
- 아트 트렌드: 다크 톤 2D 일러스트. 포스트아포칼립스 폐허 배경에 애니메이션 스타일 캐릭터. 인디 모바일 게임 전형적 아트 수준.
- 종합 코멘트: 텍스트 중심 서바이벌 로그라이크로, SAN(정신력) 시스템과 자원 품질 하락 메커니즘이 특색 있으나, 프로덕션 퀄리티와 비주얼이 인디 수준에 머물러 대중적 임팩트는 제한적. 하드코어 서바이벌 팬 대상 니치 타이틀.
