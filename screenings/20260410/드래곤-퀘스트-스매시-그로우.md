## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 로그라이크 던전 진입 → 쿼터뷰 3D 필드에서 몬스터 조우 및 실시간 액션 전투(가상 조이스틱 + 스킬 버튼) → 제한시간 내 클리어 → 보상 획득 → 캐릭터 강화/편성 → 다음 챕터 진행. AUTO 전투 지원.
- UI / 연출 인상: 토리야마 아키라 원화 기반의 드퀘 특유의 캐릭터 디자인이 3D로 재현됨. 세로화면(포트레이트) 모바일 최적화. 상단에 공지/선물/미션/이벤트 배너 다수 배치, 하단에 스토리/편성/공방/교환소/뽑기 탭 구성. 재화(골드, 스태미나 225/150)와 전투 UI(HP바, 타이머, 스킬 아이콘)가 일반적인 모바일 RPG 문법을 따름. CBT 기념 미션 표시로 아직 클로즈드 베타 단계로 보임.
- 메타 시스템 흔적: 편성(파티 구성), 뽑기(가챠), 교환소(상점), 모험의 서(도감/수집), 탐험대(파견), 스매시그로우 티켓(이벤트 재화) 등 전형적인 모바일 RPG 수익화/메타 시스템 구비.

## 평가
- 트렌드 적합성: 낮음. 콘솔 IP(드래곤 퀘스트)의 모바일 포팅/스핀오프로, 독자적 모바일 네이티브 트렌드와는 거리가 있음. 로그라이크 요소를 접목했으나 IP 의존도가 높아 트렌드 참고 가치가 제한적. 콘솔 IP 이식작은 트렌드 적합성에서 감점 대상.
- 프로덕션 퀄리티: 높음. 스쿠에어 에닉스 수준의 3D 캐릭터 모델링, 토리야마 아키라 아트 스타일의 충실한 재현, 이펙트 연출이 우수. 대형 퍼블리셔답게 UI 완성도와 전반적 폴리싱이 높음.
- 게임 깊이: 중간. 로그라이크 던전 탐색에 제한시간 메커니즘, 파티 편성, 스킬 조합 등의 전략 요소가 있으나 AUTO 전투 지원과 간소화된 조작(가상 조이스틱 + 단순 스킬)으로 캐주얼에 가까움. 가챠·스태미나 기반의 전형적 모바일 RPG 구조.
- 아트 트렌드: 중간. 드래곤 퀘스트 고유의 클래식 일본 RPG 아트 스타일은 높은 퀄리티지만, 현재 모바일 시장의 아트 트렌드(스타일라이즈, 2D 일러스트 중심, 미니멀)와는 방향이 다름.
- 종합 코멘트: 스쿠에니의 드래곤 퀘스트 IP를 활용한 모바일 로그라이크 액션 RPG. 프로덕션 퀄리티는 높으나, 콘솔 IP 이식작 특유의 보수적 게임 디자인과 전형적인 모바일 RPG 수익화 구조(가챠, 스태미나, 이벤트 재화)를 답습하고 있어 독자적인 게임 디자인 트렌드 레퍼런스로서의 가치는 제한적. CBT 단계로 향후 변화 여지가 있으나 현재 관찰 기준으로는 IP 팬층 타겟의 안전한 운영형 게임.
- 선정 여부: 후보 풀
