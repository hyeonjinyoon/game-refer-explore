## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 위자드 캐릭터를 조이스틱으로 이동하며 자동 발사되는 마법으로 몰려드는 몬스터를 처치. LEVEL/Wave 시스템으로 난이도 상승. 경험치 아이콘(초록 보석) 수집으로 레벨업, 킬 카운트 표시. 전형적인 서바이버류 "이동 + 자동공격 + 레벨업 선택" 루프.
- UI / 연출 인상: 어두운 보라색 톤 배경에 2D 카툰 스타일 캐릭터/몬스터. HUD는 좌측 상단에 HP바, 레벨, 웨이브, 킬/보석 카운트. 상단 중앙에 보스급 포탈 오브젝트 배치. 좌측 하단 보라색 반투명 조이스틱. UI가 깔끔하고 가독성 양호하나, 전체적으로 단조로운 비주얼. 메인 메뉴(02.png)에서 PLAY/HEROES/UPGRADES/SETTINGS 4버튼 구조 확인.
- 메타 시스템 흔적: 메인 메뉴에 HEROES(캐릭터 수집/선택), UPGRADES(영구 강화) 메뉴 존재. 우측 상단 재화 아이콘(코인 10개) 확인. 웨이브 클리어 후 메타 진행 구조로 추정.

## 평가
- 트렌드 적합성: 뱀파이어 서바이버류는 이미 레드오션 단계에 진입한 장르. 위자드/마법 테마는 차별점이 약하며, 장르 자체의 신선도가 크게 떨어진 상태. 모바일 전환 자체는 여전히 수요가 있으나, 독자적 메커니즘이 보이지 않음.
- 프로덕션 퀄리티: 중하. 캐릭터와 몬스터 스프라이트가 단순한 2D 카툰풍이며, 이펙트와 화면 연출이 소박함. UI 배치는 기능적으로 무난하나 세련되지 않음. 인디~소규모 개발 수준.
- 게임 깊이: 낮음. 캡처 기준 LEVEL 1 W1 단계에서 이미 핵심 루프가 반복적. HEROES/UPGRADES 메타가 있으나 영상에서 확인 가능한 범위에서는 깊이를 더하는 고유 시스템이 드러나지 않음.
- 아트 트렌드: 심플 카툰 2D. 현재 모바일 시장에서 경쟁력 있는 아트 수준은 아님. 뱀서류 원작(Vampire Survivors)의 미니멀 픽셀 아트와도 차별화 부족, 고퀄리티 서바이버류(Survivor.io 등)와도 격차 존재.
- 종합 코멘트: Vampire Survivors 포뮬러를 위자드 테마로 모바일 이식한 전형적 팔로워 타이틀. 독자적 메커니즘이나 아트 방향성에서 차별점이 부족하며, 이미 포화된 서바이버류 시장에서 눈에 띄기 어려운 수준의 프로덕션. 참고 가치 낮음.
