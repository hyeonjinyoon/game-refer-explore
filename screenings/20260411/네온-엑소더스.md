## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 사이버펑크 세계관의 탑다운 뷰에서 다수의 적을 처치하며 생존. 적 처치 시 경험치 획득 → 레벨업 시 스킬/무기 선택 → 점진적 파워 스케일링. 12분 제한 타이머 존재. 0.3 지점(Lv.10, Kills:218) → 0.5 지점(Lv.18, Kills:559) → 0.7 지점(Lv.27, Kills:1049)으로 킬 수와 레벨이 꾸준히 상승하는 전형적 뱀서 커브.
- UI / 연출 인상: 네온 색감(시안, 오렌지, 핑크)을 활용한 사이버펑크 테마. HP바와 경험치바가 상단에 배치, 하단에 장착 스킬 아이콘 + 조이스틱. 폭발 이펙트가 화려하며(0.5 지점에서 대규모 오렌지 폭발), 적 밀도가 후반부로 갈수록 크게 증가. 전반적으로 깔끔한 모바일 최적화 UI. 안드로이드 네비게이션바가 하단에 노출.
- 메타 시스템 흔적: 하단 스킬 슬롯에 숫자 표시(업그레이드 레벨), 패시브 스킬 행이 별도 존재(2행 구성). 0.7 지점에서 스킬 슬롯이 거의 꽉 찬 상태. 골드(좌하단 숫자 108→268→529) 확인 — 런 외부 영구 성장 시스템 존재 가능성 높음.

## 평가
- 트렌드 적합성: 뱀서라이크 장르의 모바일 변주. 사이버펑크 테마를 입혀 차별화 시도. 장르 자체는 성숙기에 접어들었으나 모바일에서는 여전히 수요 존재.
- 프로덕션 퀄리티: 중급. 이펙트와 색감 처리는 양호하나, 캐릭터/적 스프라이트가 소형이고 디테일이 제한적. 전체적으로 인디~소규모 스튜디오 수준.
- 게임 깊이: 스킬 조합과 레벨업 선택이 핵심 의사결정. 패시브/액티브 2행 구조로 빌드 다양성 확보 시도. 골드 시스템으로 영구 성장 요소를 가미한 것으로 보이나, 12분 런 구조상 깊이에 한계 있을 수 있음.
- 아트 트렌드: 네온 사이버펑크 팔레트(다크 블루 배경 + 네온 시안/오렌지/핑크 하이라이트). 도트 기반 스프라이트. 시각적으로 통일감 있으나 최신 프리미엄 타이틀 대비 아트 퀄리티 격차 존재.
- 종합 코멘트: Vampire Survivors 포맷에 사이버펑크 스킨을 입힌 전형적 팔로워. 이펙트 연출과 색감은 준수하지만, 장르 내 차별화 포인트가 뚜렷하지 않음. 모바일 조작 최적화는 잘 되어 있으며, 가벼운 세션 플레이에 적합한 구조.
