## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 마을(기지) 건설·확장 → 히어로 편성 → 자동 전투 스테이지 진행 → 보상 획득 → 마을 업그레이드의 순환 구조. 하단 "스토리" 버튼으로 캠페인 모드 진입, 전투는 완전 자동(타이머 기반 스킬 발동).
- UI / 연출 인상: 도트(픽셀) 아트 기반의 세로 화면 구성. 마을 화면은 아이소메트릭 뷰로 건물·캐릭터·자원 오브젝트 배치. 전투 화면은 사이드뷰 액션으로 전환되며, HP 바·데미지 숫자·스킬 이펙트가 표시됨. 전반적으로 깔끔한 레트로 감성이나 UI 요소(버튼, 폰트)는 범용적 모바일 방치형 느낌.
- 메타 시스템 흔적: 마을 건설 시스템(건물 레벨업), 히어로 수집·육성, 스토리/캠페인 진행, 자원(재화) 관리. 상단에 구글플레이 광고 ID 워터마크가 노출되어 개발 초기~소프트런칭 단계로 추정.

## 평가
- 트렌드 적합성: 방치형 RPG + 마을 건설 조합은 검증된 공식이나 차별화 요소가 뚜렷하지 않음. 최근 트렌드인 로그라이크·전략 깊이 요소는 영상에서 확인 어려움.
- 프로덕션 퀄리티: 중하. 픽셀 아트 자체는 일정 수준이나 UI 완성도·이펙트 다양성이 부족. 워터마크 노출 등 미완성 느낌.
- 게임 깊이: 영상 기준 마을 건설 + 자동 전투의 단순한 루프. 전략적 선택지나 플레이어 개입 여지가 제한적으로 보임.
- 아트 트렌드: 픽셀/도트 아트 스타일은 인디 감성으로 꾸준한 수요가 있으나, 최근 하이퀄리티 픽셀 아트 기준에는 미치지 못함.
- 종합 코멘트: 마을 건설 + 방치형 자동 전투의 전형적 조합. 픽셀 아트 감성은 있으나 프로덕션 완성도와 차별화 포인트가 부족하여 레퍼런스 가치는 제한적.
