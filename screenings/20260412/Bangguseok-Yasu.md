## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: TradingView 스타일 캔들 차트를 보며 BUY/SELL 버튼으로 주식을 매수/매도하는 단타 트레이딩. DAY 단위(14일 제한)로 진행되며, PNL(손익)과 BALANCE(잔고)를 관리하며 생존하는 서바이벌 구조.
- UI / 연출 인상: 다크 테마 기반의 깔끔한 트레이딩 UI. TradingView 차트 위젯을 그대로 활용한 듯한 캔들 차트가 중앙에 위치. AVG(평균단가) 라인, 현재가 표시 등 실제 트레이딩 앱과 유사한 구성. 하단에 BUY(초록)/SELL(빨강) 대형 버튼 배치로 직관적 조작.
- 메타 시스템 흔적: FREE CASH(광고 시청으로 +100,000원 획득), MARKET MAKER(1M 비용으로 고래 소환), Analyst Report(50k - BUY/Vol:HIGH 등 정보 제공), Insider Info(100k - BRIBE로 내부 정보 매수) 등 다양한 수익화/메타 요소 존재. 포지션 수량 제한(Max) 시스템도 확인.

## 평가
- 트렌드 적합성: 주식/투자 관심 트렌드와 맞물린 소재. MZ세대 주식 열풍과 단타 문화를 게임화한 시의적절한 컨셉. 서바이벌(14일 제한) 요소로 긴장감 부여.
- 프로덕션 퀄리티: 중간. TradingView 차트 통합으로 시각적 신뢰감은 높으나, 전반적 UI는 단순한 편. 모바일 세로 화면에 최적화된 레이아웃.
- 게임 깊이: 중간. BUY/SELL의 단순 조작이지만, Analyst Report/Insider Info 등 정보 구매 시스템, Market Maker(고래) 소환 등으로 전략적 깊이를 더함. 14일 서바이벌 구조가 회차별 리플레이 동기 부여.
- 아트 트렌드: 다크 모드 트레이딩 앱 스타일. 실제 증권 앱을 모방한 사실적 UI 디자인. 캐주얼 게임보다는 시뮬레이션 지향.
- 종합 코멘트: 주식 단타를 게임화한 니치 장르. 실제 트레이딩 경험을 단순화하면서도 서바이벌 요소와 메타 시스템(광고 보상, 정보 매수 등)으로 모바일 게임성을 확보. 수익화 구조(광고 시청, 인앱 재화)도 자연스럽게 녹아 있음.

## 심층 분석

### 영상 심층 시청
- 초/중/후반부 캡처 기반 게임플레이 진행 흐름:
  - 초반(~10s): 메인 트레이딩 화면. FREE CASH(광고 W100,000), MARKET MAKER(SUMMON W1M), Analyst Report(W50k, GET 버튼), Insider Info(W100k, BRIBE 버튼)이 모두 활성화 상태. 자금 확보와 정보 수집 단계.
  - 중반(~90s): Analyst Report와 Insider Info가 "BLOCKED" 상태로 변경됨. 한번 사용하면 쿨다운이 걸리는 구조. 제한된 정보 속에서 판단해야 하는 긴장감 조성.
  - 후반(~160s): Analyst Report가 "BUY" 신호 + "Vol: HIGH 77%" 표시. 차트 분석 결과를 바탕으로 매수 타이밍을 잡는 상황. Insider Info는 BRIBE 상태 유지.
- 메타 시스템 (육성/수집/뽑기/스킬 트리 등):
  - 레벨 1~10 단계별 서바이벌: 각 레벨마다 목표 금액 달성 필요. 파산 시 100만원부터 재시작.
  - 아이템 시스템: 세력 소환(Market Maker), 찌라시(Analyst Report), 내부자 정보(Insider Info), 무료 충전(광고 시청)
  - 랭킹 시스템 예정: 전 세계 유저 간 수익률 경쟁
  - 20여 가지 실제 차트 패턴 구현: 쌍바닥, 헤드앤숄더, 골든크로스, 데드캣 바운스 등
- UI 흐름: 다크 테마 기반 세로 모드. 상단에 4개 기능 패널(FREE CASH / MARKET MAKER / Analyst Report / Insider Info)이 고정 배치. 중앙에 캔들 차트, 하단에 BUY/SELL 버튼. 실제 증권 앱과 유사한 직관적 구성.
