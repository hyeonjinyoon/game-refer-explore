## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 턴제 자동 전투(回合 1/20 형태, 최대 20턴) + 캐릭터 소환(喚靈) + 육성. 사이버펑크풍 도시 배경의 횡스크롤 전투 화면에서 치비 캐릭터들이 자동으로 교전하며, 배속(X1) 및 스킵 버튼 제공. 전형적인 방치형 RPG 구조.
- UI / 연출 인상: 중화권 모바일 RPG의 전형적 UI 레이아웃. 소환 화면에서 고퀄리티 일러스트(Live2D 추정)로 캐릭터를 어필. 전투 화면은 2D 횡스크롤로 배경 아트 퀄리티가 양호하나 전투 연출 자체는 단순. 중국어(번체) 기반 대만 서버 버전.
- 메타 시스템 흔적: 가챠(普通喚靈/進階喚靈, 48회 내 SSR 확정), 온라인 보상(在線獎勵, 5/10/20/40/60분 타이머), IAP 상점(喚靈禮包 $4.99~$49.99, 최대 1100% 반환 표기), 주간 한정 구매 제도.

## 평가
- 트렌드 적합성: 미소녀 수집형 RPG는 포화 장르이나 꾸준한 수요 존재. 방치형 + 턴제 조합은 중화권에서 검증된 포맷. 글로벌 차별화 요소는 뚜렷하지 않음.
- 프로덕션 퀄리티: 캐릭터 일러스트는 상급(소환 화면 기준), 전투 연출은 중~중하급. UI는 기능적이나 세련미 부족. 전체적으로 중소 규모 개발사의 평균적 수준.
- 게임 깊이: 자동 전투 + 가챠 중심 구조로 전략적 깊이는 제한적. 20턴 제한 전투, 진용 추천(陣容推薦) 기능 존재하나 플레이어 개입 여지가 적어 보임.
- 아트 트렌드: 판타지 + 사이버펑크 혼합 세계관, 미소녀 캐릭터 디자인. 아트 스타일은 현재 중화권 모바일 게임 트렌드에 부합하나 독창성은 약함.
- 종합 코멘트: 중국 개발사(杭州碧游信息技术有限公司)의 전형적인 미소녀 수집형 방치 RPG. 2026년 4월 9일 대만 서버 출시. 가챠와 과금 구조가 공격적(1100% 반환 마케팅)이며, 게임플레이 차별화 요소가 부족. 레퍼런스 가치는 가챠/과금 설계 패턴 참고 정도.
- 선정 여부: 후보 풀
