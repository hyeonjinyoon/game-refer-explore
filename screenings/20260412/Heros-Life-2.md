## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 웨이브 기반 서바이벌 — 필드에서 자동/반자동 공격으로 슬라임 등 몬스터를 처치하며 경험치(젬) 수집 → 레벨업 시 스킬 선택/강화 → 20웨이브 클리어 목표. 조이스틱 이동 + 자동 공격 구조.
- UI / 연출 인상: 세로 화면에 최적화된 깔끔한 2D 픽셀/카툰 아트. 상단에 웨이브 타이머, 레벨/경험치 바, 스킬 슬롯(최대 8~9개) 배치. 골드/재화 표시 우측 상단. 화면 전반이 시인성 좋고 이펙트가 화려하지만 과하지 않음.
- 메타 시스템 흔적: 스킬 슬롯에 레벨 표기(LV1~MAX), 재화(골드·보석) 2종 확인. 캐릭터 수집/교체 시스템 존재 추정(캡처 간 캐릭터 변경). 튜토리얼 모드와 일반 웨이브 모드 분리.

## 평가
- 트렌드 적합성: Vampire Survivors 계열 서바이벌 로그라이크가 모바일에서 여전히 강세. 세로화면 채택으로 캐주얼 접근성 높임. 국산 타이틀로서 한국 시장 타겟 명확.
- 프로덕션 퀄리티: 중상급. UI 완성도와 이펙트 품질이 준수하며, 캐릭터/몬스터 디자인 통일감 있음. 출시 전 빌드치고는 안정적인 인상.
- 게임 깊이: 스킬 8~9종 조합, 레벨 36단계, 20웨이브 구조로 1회 플레이 내 빌드 다양성 확보. 다만 장기 메타(캐릭터 수집, 영구 강화 등)는 영상만으로 파악 제한적.
- 아트 트렌드: 귀여운 SD 캐릭터 + 밝은 톤 필드 배경. 한국 캐주얼 모바일 시장에서 선호되는 스타일. Survivor.io, 매직서바이벌 등과 유사한 비주얼 톤.
- 종합 코멘트: 검증된 서바이벌 로그라이크 공식을 세로화면 + 국산 감성으로 재해석한 타이틀. 스킬 조합 깊이와 UI 완성도가 인상적이며, 캐주얼 유저 접근성이 높아 보임. 출시 후 메타 시스템 깊이에 따라 리텐션 결정될 것.
- 선정 여부: TOP3
- 최종 ELO: 1594

## 심층 분석

### 영상 심층 시청
- 초/중/후반부 캡처 기반 게임플레이 진행 흐름:
  - 초반(~1분): 튜토리얼 모드. "Tutorial 02:29" 타이머 표시. 레벨업 시 스킬(화살)/버프(심장·체력+15%)/보급(포션·체력12%회복) 중 택1 선택 UI. LV7/36, 골드 0, 경험치 51. 스킬 선택 UI가 두루마리 형태로 세로화면에 최적화.
  - 중반(~5분): 본격 전투(Battle) 모드 진입. Wave 2/20, LV4/36. 필드에서 슬라임 등 몬스터를 자동공격으로 처치하며 이동. 스킬 슬롯 4~5개 장착 상태. 경험치 젬 수집 구조 확인. 배경이 초록 들판으로 변경됨.
  - 후반(~10분): Wave 15/20, LV29/36, 골드 343, 경험치 1.1K. 스킬 슬롯 8~9개 가득 참(MAX 레벨 스킬 다수). 화면 전체에 다양한 투사체/이펙트가 날아다니며 화려한 전투. 동료 캐릭터(흰색 SD캐릭터)도 함께 전투 참여.
- 메타 시스템 (육성/수집/뽑기/스킬 트리 등):
  - 캐릭터 수집 시스템: 최대 4명 파티 편성, 가챠(뽑기)로 캐릭터 획득. 영상 후반 12:06에 캐릭터 목록 확인.
  - 스킬 조합: 레벨업마다 3개 중 택1로 스킬/버프/보급 선택. 최대 8~9슬롯까지 빌드 구성.
  - 유물 시스템: 스킬 외 유물(Relic) 조합으로 시너지 전략 추가.
  - 재화: 골드(인게임) + 보석(프리미엄) 2종 구조.
  - 영구 강화: 스테이지 클리어 후 골드로 캐릭터 업그레이드 추정.
  - BM: 캐릭터 가챠 중심. 한국 정식 출시 후 BM 확인 필요.
- UI 흐름:
  - 세로화면 전용 UI. 상단에 Wave/타이머, 레벨/경험치 바, 재화(골드·보석) 표시.
  - 스킬 슬롯은 상단 좌측에 아이콘+레벨 형태로 배치(최대 8~9개).
  - 레벨업 선택 UI는 세로 두루마리 3장 형태로 직관적.
  - 하단 조이스틱 이동 + 자동 공격 구조. 일시정지 버튼 좌상단.
  - 전반적으로 깔끔하고 시인성 높은 2D 아트 스타일.
