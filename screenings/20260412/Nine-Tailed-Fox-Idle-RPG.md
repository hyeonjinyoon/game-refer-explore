## 게임플레이 관찰 (절차 4번 캡처 기반)
- 핵심 루프: 스테이지 기반 자동 전투 → 보상 획득(EXP, 장비) → 장비 소환/강화 → 다음 스테이지 진행. 전형적인 방치형 RPG 루프. 성장 던전 2단계 등 별도 던전 콘텐츠 존재. 보스 도전하기 버튼으로 보스 스테이지 분리.
- UI / 연출 인상: 세로 화면 최적화된 레이아웃. 하단 5탭 메뉴(성장/장비/스킬/콘텐츠/상점). 우측에 퀘스트/패스/출석/영입 등 사이드 버튼 밀집. 데미지 넘버가 화면에 다수 표시(5.24A 등 약식 표기). 스테이지 클리어 연출은 골든 이펙트로 화려하지만 정형화된 느낌. 채팅 UI 하단에 존재.
- 메타 시스템 흔적: 장비 소환(가챠) 시스템 - D~DDD+ 등급 체계, 1회/10회/50회 뽑기 + 광고 시청 무료 소환. 우편함/패스/이벤트/컬렉션 등 전형적인 모바일 RPG 메타 시스템. AUTO 버전 전투, 무기 강화 횟수 카운트(10/10) 확인.

## 평가
- 트렌드 적합성: 세로형 방치 RPG는 현재 레드오션 장르이나 꾸준한 수요 존재. 구미호/한국 설화 테마는 차별점이 될 수 있으나 게임플레이 자체는 기존 방치형과 대동소이.
- 프로덕션 퀄리티: 중급. 2D 스프라이트와 이펙트는 깔끔하나 눈에 띄는 혁신 없음. UI 구성은 안정적이지만 우측 버튼 과밀집 문제.
- 게임 깊이: 장비 등급 세분화(D~DDD+), 스킬 슬롯 6개, 성장 던전 등 콘텐츠 레이어 존재하나, 자동 전투 중심이라 전략적 깊이는 제한적.
- 아트 트렌드: 한국식 2D 애니메이션 스타일. SD 캐릭터 기반 전투 화면. 구미호 컨셉에 맞는 동양풍 아트. 전체적으로 무난하지만 독보적이지는 않음.
- 종합 코멘트: 전형적인 세로형 방치 RPG로 기존 공식을 충실히 따르고 있음. 구미호 테마와 국산 IP라는 점이 차별점이나, 게임플레이/시스템 면에서 기존 방치형 대비 눈에 띄는 혁신은 부족. 타겟 유저층(캐주얼 방치형 선호)에게는 적합.
- 선정 여부: 후보 풀
- 최종 ELO: 1502
